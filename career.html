Month 1: HTML & CSS Foundations

Build the structural and stylistic backbone of web pages.

  • Learn semantic HTML5 tags.
  • Master CSS selectors, Flexbox, and Grid.
  • Build a personal portfolio website.
Month 1

Month 2: JavaScript Fundamentals

Bring interactivity and logic to your websites.

  • Understand variables, data types, and functions.
  • Learn DOM manipulation and event handling.
  • Complete a simple JavaScript project like a to-do list app.
Month 2

Month 3: Intro to Frontend Frameworks

Learn modern tools for building complex user interfaces.

  • Choose a framework (React, Vue, or Angular).
  • Learn about components, state management, and props.
  • Build a single-page application (SPA).
Month 3

Month 4: Backend Development with Node.js

Build the server-side logic and APIs for your applications.

  • Learn Node.js and Express framework.
  • Understand RESTful APIs and how to build them.
  • Connect your application to a database (MongoDB).
Month 4

Month 5: Databases & Full Stack Integration

Master data storage and connect your frontend and backend.

  • Deep dive into database concepts (SQL/NoSQL).
  • Implement user authentication and authorization.
  • Deploy a full-stack MERN/MEVN application.
Month 5

Month 6: Deployment & Advanced Topics

Go live with your projects and learn advanced skills.

  • Learn about deployment platforms (Vercel, Netlify, Heroku).
  • Understand version control with Git and GitHub.
  • Explore topics like testing, CI/CD, and web performance.
Month 6
